#  Modelagem de Banco de Dados: Saneamento Cidade Inteligente

Este repositório contém os entregáveis da disciplina de Modelagem de Banco de Dados, focados no projeto de Saneamento para Cidades Inteligentes.

## ✅ Modelos Entregues

* **Modelo Conceitual:** Representação das entidades e relacionamentos principais do sistema.
    * *Arquivo:* `Conceitual_saneamento.brM3`
* **Modelo Físico:** Script SQL para a criação das tabelas no ambiente PostgreSQL.
    * *Arquivo:* `Projeto_Fisico_Saneamento.sql`
